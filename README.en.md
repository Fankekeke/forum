# 基于Spring学习论坛系统

#### Description
（1）用户的注册登录
（2）帖子的发布与评论
（3）前台不同学科帖子的分类与展示
（4）搜索功能，对于想要搜索的学科与具体帖子进行搜索
（5）基于用户喜欢的帖子的类型进行相关学科类别的帖子进行内容推荐
（6）用户对于自己帖子的管理
（7）用户对于自己喜欢的帖子点赞及收藏
（8）对敏感词汇的屏蔽处理
（9）管理员对用户帖子以及评论的管理
（10）管理员对于帖子的板块分类管理
